## Color Scheme (Dark Theme)
- Background: Dark (black/zinc-950)
- Text: Light (white/zinc-50)
- Muted text: zinc-400
- Primary buttons: Primary color with hover effects
- Secondary buttons: Outlined with hover effects
- Cards: Subtle border with shadow
- Icon backgrounds: Primary color with 10% opacity

## Responsive Behavior
- **Mobile (< 768px)**: 
  - Single column layout
  - Stacked buttons
  - Feature cards in 1 column

- **Tablet (768px - 1024px)**:
  - Feature cards in 2 columns
  - Side-by-side buttons

- **Desktop (> 1024px)**:
  - Feature cards in 4 columns
  - Full-width container with max-width
  - Optimal spacing and padding
