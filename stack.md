# Stack

## Neon
Neon is a serverless Postgres database platform that separates storage and compute, offering instant scaling, branching capabilities for development workflows, and a generous free tier. It's designed for modern applications with features like automatic backups and point-in-time restore.

## shadcn/ui
shadcn/ui is a collection of beautifully designed, accessible, and customizable React components built with Radix UI and Tailwind CSS. Unlike traditional component libraries, you copy the components into your project, giving you full control and ownership of the code.

## Drizzle ORM
Drizzle ORM is a lightweight TypeScript ORM that provides a type-safe way to interact with databases. It offers both SQL-like query builders and relational queries, excellent TypeScript inference, and minimal runtime overhead while maintaining great developer experience.

## Clerk
Clerk is a complete authentication and user management solution that provides pre-built UI components, secure session management, and support for multiple authentication methods including social logins, magic links, and multi-factor authentication. It simplifies adding auth to your application with minimal code.

## Zod
Zod is a TypeScript-first schema validation library that allows you to build schemas for data validation and type inference. It provides runtime validation with excellent TypeScript integration, ensuring your data matches expected types at both compile-time and runtime.
